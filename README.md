# WebStore
Опис проекту: WebStore – це інтернет-магазин, який містить каталог товарів, котрий дає можливість продавати різноманітні товари користувачам за допомогою мережі Інтернет.

Технічне завдання:

Головний інтерфейс: веб-сайт повинен мати каталог товарів, кабінет адміністратора, особистий кабінет користувача з корзиною для додавання обраних товарів, а також вікно для пошуку товарів. 

Зберігання інформації: для зберігання переліку товарів інтернет-магазину та особистих даних користувачів у проекті повинна бути застосована база даних SQL з використанням технології Entity Framework.

Місце розташування: веб-сайт має бути розташований на платформі хмарних обчислень Microsoft Azure.
